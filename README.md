# 🎮 Gaming Gear E-commerce 'PULSE'

게이밍 용품을 판매하는 **이커머스 웹 프로젝트** 이며, 브랜드는 **PULSE** 입니다.

---

## 1. 프로젝트 소개

**게이밍 용품을 판매하는 이커머스**를 주제로 한 팀 프로젝트입니다.
**React**를 베이스로 구현하였으며, **API기반**으로 회원가입/로그인, 상품 목록 조회, 카테고리별 상품 탐색, 상품 상세 조회, 장바구니 기능, 위시리스트, 구매처리까지 실제 이커머스의 핵심 흐름을 모두 담아 내는 것을 목표로 하였습니다.

### 주요 목표

- API 데이터 처리
- 인증 기능 구현 (회원가입/로그인)
- Zustand를 활용한 전역 상태관리
- router를 활용한 SPA 라우팅
- 상품 목록 및 상세 조회
- 장바구니 및 위시리스트 기능
- 마이페이지 제공
- 반응형 UI 구현
- 사용자 중심 UX

## 6. 주요 기능

### 1) 회원가입 / 로그인

- 아이디 이메일 형식 검증
- 비밀번호 유효성 검사
- 아이디 중복 검사
- 이용 약관 동의
- 로그인 성공 시 메인 페이지 이동
- 로그인 상태에 따라 헤더 UI 변경
- API데이터를 이용한 관리자 로그인

### 2) 메인 페이지

- API데이터를 이용한 상품 목록 출력
- 서비스 로고 및 카테고리 메뉴 제공
- 히어로 배너 영역 구성
- 테마 토글 제공 (라이트/다크)

### 3) 카테고리 목록 페이지

- API데이터를 이용한 상품 목록 출력
- 카테고리에 맞는 상품만 출력
- 2차 카테고리별 상품 목록 필터링 제공
- 상태 및 가격에 따른 직관적인 필터링 제공
- 조건에 맞는 상품에 따라 페이지네이션 적용
- 위시리스트 기능 제공 (Zustand)
- 장바구니 담기

### 4) 상품 상세 페이지

- API데이터를 이용한 추천 상품 목록 출력
- 선택된 상품 이미지, 이름, 가격, 기본정보, 배송정보 출력
- 수량 조절
- 장바구니 담기
- 구매하기

### 5) 장바구니 페이지

- API를 이용한 추천 상품 목록 출력
- Zustand로 전역 상태 관리
- 담긴 상품 리스트 확인
- 수량 조절
- 개별 삭제 / 전체 삭제
- 총 상품 금액 및 결제 예정 금액 자동 계산
- 구매하기

### 6) 마이 페이지

- Zustand 전역 상태 관리
- 프로필
- 주문내역
- 위시리스트
- 리뷰
- 쿠폰
- 배송지

### 7) 404 페이지

- 존재하지 않는 경로 접근 시 안내 페이지 표시
- 메인 페이지로 이동 버튼 제공

### 8) 반응형 레이아웃

- Mobile / Tablet / PC 환경 대응
- 화면 크기에 따라 상품 카드 및 레이아웃 변경

---

## 7. 페이지 구성

- `/` : 메인 페이지
- `/login` : 로그인 페이지
- `/categories` : 전체 상품 목록 페이지
- `/categories/:categoryName` : 카테고리별 상품 목록 페이지
- `/product/:id` : 상품 상세 페이지
- `/shoppingcart` : 장바구니 페이지
- `/mypage` : 마이 페이지
- `/brand` : 브랜드 페이지
- `*` : 404 페이지
